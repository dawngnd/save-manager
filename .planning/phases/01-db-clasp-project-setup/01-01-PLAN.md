<task id="01-01-T2">
<objective>Tạo Code.js với getSpreadsheet() helper và initializeSheets() function</objective>
<read_first>
- .planning/phases/01-db-clasp-project-setup/01-RESEARCH.md (section 1: Schema, section 4: Script Properties)
- .planning/REQUIREMENTS.md (DB-01: schema definition)
- .planning/phases/01-db-clasp-project-setup/01-CONTEXT.md (D-02: connect via SpreadsheetApp.openById)
</read_first>
<action>
Tạo file `backend/Code.js` chứa:

**Function `getSpreadsheet()`:**
- Đọc property `SPREADSHEET_ID` từ `PropertiesService.getScriptProperties()`
- Throw Error với message `"SPREADSHEET_ID not configured in Script Properties"` nếu property không tồn tại
- Return `SpreadsheetApp.openById(ssId)`

**Function `initializeSheets()`:**
- Gọi `getSpreadsheet()` lấy spreadsheet object
- Kiểm tra sheet `Users`: nếu chưa tồn tại thì tạo bằng `insertSheet('Users')`. Nếu row 1 trống thì set header: `['username_bankcode']`
- Kiểm tra sheet `Deposits`: nếu chưa tồn tại thì tạo bằng `insertSheet('Deposits')`. Nếu row 1 trống thì set header: `['id', 'amount', 'interest_rate', 'status', 'expected_interest', 'created_at', 'maturity_at', 'user_bankcode']`
- Return object `{ users: usersSheet, deposits: depositsSheet }` hoặc log thông báo thành công

Sử dụng `getSheetByName()` để kiểm tra sheet tồn tại, `getRange(1, 1, 1, numCols).getValues()` để kiểm tra header row trống, và `setValues()` để ghi headers.
</action>

<action>
Tạo file `backend/SETUP.md` chứa hướng dẫn:

1. **Tạo Google Sheet**: Truy cập sheets.google.com, tạo spreadsheet mới, đặt tên "Save Manager DB"
2. **Cấu hình timezone**: Vào File > Settings > Time zone, chọn `(GMT+07:00) Ho Chi Minh`
3. **Lấy Spreadsheet ID**: Copy ID từ URL `https://docs.google.com/spreadsheets/d/{SPREADSHEET_ID}/edit`
4. **Cấu hình Script Properties**: Mở GAS editor (hoặc `clasp open`), vào Project Settings > Script Properties, thêm key `SPREADSHEET_ID` với value là ID vừa copy
5. **Chạy initializeSheets()**: Trong GAS editor, chọn function `initializeSheets` và chạy để tự động tạo headers

Lưu ý quan trọng: timezone `appsscript.json` và Google Sheet settings là ĐỘC LẬP, phải set riêng cả 2 nơi.
</action>
