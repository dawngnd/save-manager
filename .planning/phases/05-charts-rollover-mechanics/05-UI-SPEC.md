## Visuals & Hierarchy

- **Primary Focal Point:** Biểu đồ dự phóng tăng trưởng tài sản (khi hiển thị qua query param) hoặc danh sách các khoản gửi đã đáo hạn.
- **Visual Hierarchy:** Số dư tổng kết -> Biểu đồ tăng trưởng -> Danh sách khoản gửi cần xử lý -> Nút FAB "+".
- **Accessibility:** Nút bấm hành động đều có nhãn text (không dùng icon đơn độc).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#17212b` | Background, surfaces |
| Secondary (30%) | `#0e1621` | Cards, sidebar, nav, bottom sheets |
| Accent (10%) | `#5288c1` / `#64b5f6` | `#5288c1` cho nút bấm/FAB; `#64b5f6` làm màu vẽ nét biểu đồ |
| Destructive | `#ff4d4d` | Error states, cancel/exit actions, confirmation buttons |

Accent reserved for: Primary interactive controls (e.g., FAB button, Submit buttons, chart line/fill, and highlighted status badges).
