<threat_model>
## Trust Boundaries
| Boundary | Description |
|----------|-------------|
| GitHub Actions Runner -> GitHub Repositories | Việc deploy lên GitHub Pages yêu cầu ghi đè vào nhánh gh-pages thông qua khóa bảo mật GitHub Token mặc định của runner. |
| Client Web Browser -> GitHub Pages hosting | SPA chạy trên trình duyệt người dùng được tải về từ máy chủ CDN công khai của GitHub Pages. |

## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| SEC-01 | Tampering | GitHub Actions Workflow | Medium | Mitigated | Cấu hình giới hạn quyền ghi nội dung (`permissions: contents: write`) để chỉ cho phép workflow đẩy kết quả build lên nhánh gh-pages, không có quyền can thiệp nhánh mã nguồn khác. |
| SEC-02 | Information Disclosure | Production Env Variables | Medium | Mitigated | Lưu trữ URL gọi API nhạy cảm của GAS (VITE_API_URL) dưới dạng secret trong GitHub Repository Settings và inject vào lúc build (Build-time inject) thay vì hardcode trực tiếp vào mã nguồn công khai. |
</threat_model>
